# Quick Start

A good way to start experimenting with the app is to load a sample project and play around.

## Importing a Project

Importing a project is much easier in Drafft 2, there is a button to Download a Sample project directly in the project manager screen.
Each time you click this button the sample project will load under a unique name allowing you to quickly spawn different versions to try things out.

For this tutorial flow, use:

- [The Garden of Fading Memories sample project](https://github.com/ajboni/drafft-examples/blob/main/projects/v2/the-garden-of-fading-memories.json)

This sample is also open to contributions via pull requests in the examples repository.
