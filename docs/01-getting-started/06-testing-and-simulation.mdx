# Testing and Simulation

Drafft's simulation tools help you validate branching and state transitions before export.

## Core Checks

- Dialogue branch routing (choice, condition, switch, event flow)
- Variable changes from `@set`
- Scenario-based branch reproducibility

## Suggested Test Loop

1. Load a scenario that represents your test setup
2. Run dialogue playback through expected branches
3. Confirm variable updates in your state views
4. Repeat with a different scenario for edge cases
