# Attachments

## Overview

Attachments are project documents for binary files. You can use them to store assets and supporting files alongside scripts and design content.

## Supported Types

Attachments support mixed file uploads, including:

- Images
- Video
- Audio
- PDF
- Other binary file types

## Current Limitations

- Image preview support is currently the primary built-in preview path
- Other file types are stored and can be downloaded, but may not have rich in-app preview

## Practical Uses

- Dialogue/reference image assets
- Audio references for voice or mood
- PDF briefs and design notes
- Shared binary resources for team workflows
