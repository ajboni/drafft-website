### What Are Kanban Boards?

Kanban boards are visual task trackers that let you manage work using draggable cards in columns. Drafft's built-in Kanban editor helps you break down your game into manageable steps.

![Kanban Board Example](/img/gallery/kanban-board.webp)

---

### Key Features

- **Columns:** Create as many columns as you like (`Ideas`, `Todo`, `Doing`, `Done`, etc.)
- **Cards:** Each card supports a title and description (with Markdown), plus links to other Drafft content.
- **Drag-and-Drop:** Reorder or move cards freely to reflect progress.
- **Project-Specific:** Each board belongs to a single project and is saved automatically.

---

### Common Use Cases

- Tracking quests or levels
- Managing writing/editing status of scenes
- Bug/feature checklists
- Planning sprints or milestones
