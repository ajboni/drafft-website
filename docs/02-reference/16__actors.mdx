# Actors

Actors represent characters used by scripts and dialogue content.

## What You Can Configure

- Biography text
- Structured actor properties (HJSON/JSON/TOML)
- Expression images used by `[expression]` tags

## Script-Scoped Views

Actor tooling includes actor-specific script views such as tagged/speech-focused content to help with writing and voice workflows.

## Notes

- Expression matching is case-insensitive
- Actor content can participate in exports depending on selected export sections
