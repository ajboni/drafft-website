# Canvas (WIP)

:::warning
This feature is **WIP**, **under construction**, and currently **pre-alpha**.

Behavior, document structure, and UI flows are likely to change and may break between versions.
:::

## Overview

Canvas is an early visual workspace intended for free-form node-based authoring and note organization.

## Current State

- Early node/edge workflow exists
- Note-style node support is available
- Editing behavior and schema are still evolving

## Safe Usage Guidance

- Use Canvas for experimentation, not production-critical pipelines
- Keep external backups for important Canvas documents
- Expect breaking changes while the feature matures

## Stability Expectations

Canvas is exposed for testing and feedback. Treat it as unstable until explicitly marked stable in release notes.
